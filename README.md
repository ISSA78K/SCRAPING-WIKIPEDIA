# Web Scraping con Wikipedia

Proyecto realizado en Python utilizando:
- Requests
- BeautifulSoup
- Pandas

## Objetivo
Extraer información de países y población desde Wikipedia.

## Sitio utilizado
https://es.wikipedia.org/wiki/Anexo:Pa%C3%ADses_y_territorios_dependientes_por_poblaci%C3%B3n

## Resultado
El programa genera un archivo CSV con:
- País
- Población

## Tecnologías
- Python
- BeautifulSoup
- Pandas
